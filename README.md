# 小红书文案生成器 Pro v4.0

AI驱动的小红书文案生成工具，告别模板化，实现真正的智能创作。

## ✨ 核心特性

- **5种文案风格**: 真实体验、专业测评、情感共鸣、幽默风趣、极简高级
- **5大产品类别**: 美食、水果、日用品、数码、美妆（智能识别）
- **多样化生成**: 每次生成内容都不同
- **无需登录**: 打开即用，直接生成

## 🚀 部署

### Railway部署
1. 将代码推送到GitHub
2. 在Railway导入项目
3. 自动部署完成

### 本地运行
```bash
pip install -r requirements.txt
python3 app.py
```

## 📝 使用

1. 输入产品名称（如：榴莲、面膜、耳机）
2. 选择文案风格
3. 点击生成
4. 一键复制
